# Contributing to Book Mate

Thanks for your interest in contributing!

## Quick Start

1. Fork the repo and clone: `git clone https://github.com/YOUR_USERNAME/book-mate.git`
2. Create a branch: `git checkout -b feature/your-feature`
3. Set up: `cp .env_template .env` (add your `OPENAI_API_KEY`)
4. Run: `make build && make start`
5. Make your changes and test at http://localhost:7860
6. Commit: `git commit -m "Add: feature description"`
7. Push and open a PR

## Commit Prefixes

- `Add:` - New features
- `Fix:` - Bug fixes
- `Update:` - Changes to existing features
- `Docs:` - Documentation

## What We're Looking For

- Bug fixes
- Documentation improvements
- New book formats (PDF, EPUB)
- UI/UX enhancements
- Performance improvements

Check the [GitHub Issues](https://github.com/nupsea/book-mate/issues) page for open issues. Issues labeled `good first issue` are great starting points for new contributors!

## Code Style

- Follow existing patterns
- Use type hints and docstrings
- Test your changes locally

## Getting Help

Open a GitHub Issue or Discussion!
